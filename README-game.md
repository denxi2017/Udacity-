##过河游戏
###游戏运行步骤
>打开index.html文件即可看到游戏开始，请最好保持游戏画面完整出现在浏览器窗口（已经设置不出现滚动条）。
###游戏说明
>玩家角色（每次加载五种角色中随机出现一种）出现在游戏内容下方中间，通过键盘上下左右键操控角色移动。
>游戏中有三行敌人，每一行随机在最左侧生成敌人，敌人在一行内由左至右移动，敌人初始速度按一定量值随机赋予。
>敌人到达每一行行末时自动穿过这一行然后继续从最左侧出现，速度不变；玩家角色达到左、右、下边界时自动被设定为不能再往边界方向移动。
>角色向上移动，穿越三行敌人之后，到达水池时，设定为玩家闯过该关卡并跳转到下一关卡(敌人速度再次随机生成)，角色位置自动回到初始位置，积分自动增加10分。
>角色在移动过程中碰到敌人或游戏时间耗尽（默认每关时间为20秒），则弹框提示“游戏结束!”，点击确定或者按回车键重新加载游戏，关卡、积分清零。
###一些注意
>我在engine.js文档中的143-147行添加了其余角色的图片路径。
>碰撞检测的算法并没有严格去计算，根据粗略计算和实际测试设置的碰撞参数，所以可能会对体验有所影响。
>完成关卡时，虽然速度会再次生成，但不代表下一关某一个敌人速度相比上一关会更快，因为关卡增加调整的是速度系数的增加，基础速度仍然由随机生成。
>在一些浏览器上，运行效果可能会受到影响，建议在谷歌浏览器上运行。
